概述

Toolbar类包含在Ruby中创建和操作SketchUp工具栏的方法。工具栏是按钮的集合，可用于激活自定义工具或ruby脚本。

另请参阅Command对象，以获取有关创建可从工具栏调用的“命令”的详细信息。

例子：

toolbar = UI::Toolbar.new "Test"
# 该工具栏图标的作用只是在屏幕上显示Hello World
cmd = UI::Command.new("Test") {
  UI.messagebox "Hello World"
}
cmd.small_icon = "ToolPencilSmall.png"
cmd.large_icon = "ToolPencilLarge.png"
cmd.tooltip = "Test Toolbars"
cmd.status_bar_text = "Testing the toolbars class"
cmd.menu_text = "Test"
toolbar = toolbar.add_item cmd
toolbar.show

版本：

  • SketchUp 6.0
